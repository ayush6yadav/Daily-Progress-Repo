# Day 1: Linux Basics

## 1. Navigation & File Operations
* `pwd`: Print working directory.
* `ls -la`: List all files with details (permissions, size).
* `cd /var/www/html`: Navigate to the default Apache web root.
* `mkdir -p project/test`: Create nested directories.
* `touch index.html`: Create an empty file.
* `cp source.txt dest.txt`: Copy files.
* `mv old_name new_name`: Rename or move files.

## 2. System Info
* `uname -a`: System kernel info.
* `df -h`: Check disk space usage.
* `top` or `htop`: Monitor system resources.
